CONCURRENT TCP SERVER USING THREADS

to run server: 
    1. make server
    2. ./server <server ip> <server port>

to run client:
    1. make client
    2. ./client <input file> <server ip> <server port>

to clear:
    make clean_client: to clear all the executables and object files related to client
    make clean_server: to clear all the executables and object files related to server
    make clean: to clear all the executables and object files related to both client and server
                and also the output file is deleted.
